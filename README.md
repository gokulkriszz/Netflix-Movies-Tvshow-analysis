## 📌 Project Overview

This project dives into the **Netflix Titles dataset** to uncover patterns and trends across movies and TV shows available on the platform.

The end-to-end workflow covers:
- 🧹 **Data Cleaning** — handled in Microsoft Excel
- 📊 **Interactive Dashboard** — built in Power BI

> _From raw data → clean data → actionable insights._

---

## 🛠️ Tools & Technologies

| Tool | Purpose |
|------|---------|
| 📗 Microsoft Excel | Data Cleaning & Preprocessing |
| 📊 Power BI | Dashboard & Visualization |
| 📦 Kaggle Dataset | Netflix Titles Source Data |

---

## 🧹 Data Cleaning Steps (Excel)

The raw dataset required several cleaning steps before analysis:

- ✅ Removed **duplicate records** and **blank rows**
- ✅ Standardized **date formats** and **text casing**
- ✅ Handled **missing values** in `director`, `cast`, and `country` columns

---

## 📊 Power BI Dashboard Highlights

The interactive dashboard answers key business questions:

| Visual | Insight |
|--------|---------|
| 🍩 Donut Chart | Movies vs TV Shows distribution |
| 📈 Line Chart | Content growth trend (1925–2021) |
| 🌍 Map / Bar Chart | Top 10 countries by content volume |
| ⭐ Bar Chart | Ratings distribution across titles |
